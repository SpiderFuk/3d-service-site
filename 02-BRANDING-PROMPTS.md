## Direcciones Creativas

### Dirección A: Minimalista Geométrico

**Concepto:** Formas geométricas que sugieren las capas de impresión 3D o la construcción capa por capa.

**Elementos visuales:**
- Cubo 3D con líneas de capa visibles
- Hexágonos apilados
- Formas isométricas simples

**Prompt base:**
```
Minimalist logo design for a 3D printing business called "[Printo]".

Style:
- Clean geometric shapes suggesting 3D printing layers
- Isometric cube or hexagon with visible layer lines
- Flat vector design, no gradients
- Single accent color: blue (#3B82F6)
- White background

Requirements:
- Simple enough to work as favicon (16x16px)
- Scalable vector style
- Professional and modern look
- No text in the icon version
- Must work in single color (monochrome version)

Output: Logo icon only, centered, clean white background
```

**Variaciones del prompt:**

```
[VARIACIÓN 1 - Cubo con capas]
Minimalist 3D cube logo with horizontal layer lines, isometric view, 
blue color (#3B82F6), clean vector style, white background, 
suitable for tech/3D printing business

[VARIACIÓN 2 - Hexágono apilado]
Stacked hexagon logo design, three layers creating depth effect,
minimal line art, blue accent color, modern geometric style,
white background, professional tech company aesthetic

[VARIACIÓN 3 - Forma abstracta]
Abstract geometric shape suggesting 3D construction, layered 
triangles or polygons, minimal design, blue on white, 
vector style logo for manufacturing business
```

## Prompts para Elementos Adicionales

### Patrón de Fondo (opcional)

```
Subtle geometric pattern for website background.
Repeating hexagons or isometric grid, very light gray (#F8FAFC).
Minimal, barely visible texture, professional clean look.
Seamless tile pattern, modern tech aesthetic.
```

### Íconos de Servicios

```
Set of 3 minimal icons for 3D printing services:
1. Rapid prototyping - clock with cube
2. Custom parts - gear or puzzle piece
3. 3D design - pencil with cube or CAD symbol

Style: Line icons, consistent stroke width, blue color (#3B82F6),
minimal design, match style of main logo.
White background, arranged horizontally.
```

### Imagen Hero (opcional)

```
Hero image for 3D printing website.
Clean workspace with 3D printed objects, soft lighting.
Light, airy atmosphere, white/gray tones with blue accents.
Professional photography style, shallow depth of field.
Objects: geometric shapes, mechanical parts, design prototypes.
No people, focus on the printed objects and quality.
```

## Notas para la Generación

1. **Iteración:** Generar 3-4 variaciones de cada dirección antes de elegir
2. **Simplicidad:** Preferir diseños más simples que complejos
3. **Escalabilidad:** Probar mentalmente si funcionará en 16x16px (favicon)
4. **Consistencia:** Una vez elegida una dirección, mantener ese estilo
5. **Colores:** Respetar la paleta definida (#3B82F6 azul, #1E293B gris oscuro)

---

## Ejemplo de Flujo de Trabajo

1. **Elegir nombre** de la lista o proponer uno nuevo
2. **Elegir dirección** (A, B, C, o D)
3. **Generar ícono** usando el prompt base de esa dirección
4. **Iterar** con variaciones hasta satisfacción
5. **Generar wordmark** combinando ícono + tipografía
6. **Generar versiones** (horizontal, vertical, mono)
7. **Exportar** en formatos necesarios (SVG preferido)

## Prompt Universal (Ajustable)

Si ninguna dirección convence, usar este prompt adaptable:

```
Professional logo for 3D printing business called "[Printo]".

Business context:
- Service-based company offering 3D printing
- Target audience: designers, engineers, hobbyists
- Tone: Professional, modern, accessible

Design requirements:
- Style: [ELEGIR: geometric/abstract/literal/typographic]
- Primary color: Blue (#3B82F6)
- Background: White
- Aesthetic: Clean, minimal, modern
- Must work at small sizes (favicon)
- Must work in single color

Avoid:
- Overly complex details
- Realistic 3D renders
- Gradients that don't translate to print
- Generic clip-art style
- Outdated design trends

Output: Logo icon, vector style, centered on white background
```
